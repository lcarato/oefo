USAGE PATTERN:
==============
Standard workflow:
  1. Create agent: agent = QCAgent()
  2. Run QC: result, routing, breakdown = agent.run(observation)
  3. Route: if routing == RoutingDecision.AUTO_ACCEPT: save(result)

Batch workflow:
  1. Create agent: agent = QCAgent()
  2. Process: batch_results = agent.process_batch(observations)
  3. Access: batch_results['auto_accepted'], ['flagged_for_review'], ['rejected']
